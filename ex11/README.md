# Ex11 - ES6+ Classes & Private Fields

## Objectif
Maîtriser les classes ES6+ avec champs privés, static, getters/setters.

## Instructions

Implémenter une classe `Person` complète :
- Champs privés (#name, #age)
- Getters/setters avec validation
- Méthodes statiques
- Méthodes d'instance

Implémenter une classe `Rectangle` :
- Constructor avec width, height
- Getters area, perimeter
- Méthode static fromSquare(side)
